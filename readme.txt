Для корректной работы неоходим python 3.7 и выше

Используемые библиотеки:
	1. requests
	2. json
	3. time
	4. csv
	5. datetime

Для запуска скрпита необходимо:
	1. Создать standalone-прложение для работы с VK API (https://vk.com/dev)
	2. Скопировать сервисный ключ (token)
	3. Через терминал запустить актуальную версию скрипта. Далее - следовать инструкции в скрипте.
